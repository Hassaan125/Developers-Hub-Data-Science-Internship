Data Science Internship - Developers Hub
This repository contains my completed tasks for the Data Science Internship at Developers Hub. Each project focuses on solving real-world problems using Machine Learning, Data Visualization, and Statistical Analysis.

📁 Projects Overview
Task 1: Titanic Survival Prediction
Objective: Predict the survival of passengers on the Titanic based on features like age, gender, and class.

Model: Logistic Regression.

Outcome: Identified that gender and socio-economic status were key survival factors.

Task 2: Credit Risk Analysis
Objective: Analyze bank customer data to predict the likelihood of loan defaults.

Approach: Data cleaning, handling missing values, and risk classification.

Tools: Pandas, Scikit-Learn.

Task 3: Customer Churn Prediction
Objective: Predict which bank customers are likely to leave (churn).

Model: Random Forest Classifier.

Key Insight: Achieved 90.5% Accuracy. Found that usage duration is a major churn indicator.

Task 4: Insurance Claim Prediction
Objective: Build a regression model to estimate medical insurance costs.

Algorithm: Linear Regression.

Observation: High BMI and Smoking status significantly increase insurance premiums.

Task 5: Personal Loan Acceptance Prediction
Objective: Identify customers who are likely to accept a personal loan offer.

Model: Decision Tree Classifier.

Success: Achieved 89.7% Accuracy. Discovered that call duration is the strongest predictor.

🛠️ Tech Stack & Libraries
Language: Python 3.x

Data Analysis: Pandas, NumPy
